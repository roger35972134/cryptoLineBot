# cryptoLineBot
A Searcher for Cryptocurrency informations. Made by Line Bot
虛擬貨幣查詢的 Line 機器人，資料來源為[CoinGecko](https://www.coingecko.com/zh-tw)，包含前100名的貨幣種類
## Usage
Type the symbol of Current, e.g. BTC.
